# PMI SMP

> **A clean, focused modpack built for SMP and vanilla server play.**  
> No bloat. No gameplay overhaul. Just a smoother, prettier, more social Minecraft experience - with adventure datapacks ready to enable whenever your world needs them.

---

## What is PMI SMP?
![Quiet Bay](https://cdn.modrinth.com/data/NUznJSEG/images/1e7d67f7b8687853428692e1da5f236d0cca9dfa_350.webp)

PMI SMP is a lightweight **Fabric 1.21.11** client modpack designed to be dropped onto any vanilla-compatible server with minimal friction. Every mod here earns its place: the list is deliberately short, covering **performance**, **visuals**, **quality-of-life**, and **multiplayer comfort** — and nothing else.

Want to spice up your world? A curated set of **datapacks** is included and ready to activate: new terrain, new structures, Nether and End overhauls, and small vanilla-friendly tweaks that feel like they belong.

---

## 🚀 Performance

Smooth frames and fast chunk loading are the foundation.

| Mod | What it does |
|-----|-------------|
| **Sodium** | Dramatically improved rendering engine |
| **Sodium Extra** | Additional render settings and toggles |
| **Lithium** | Server-side logic optimizations (works client-side too) |
| **FerriteCore** | Significant RAM usage reduction |
| **ImmediatelyFast** | Batches immediate-mode rendering calls |
| **EntityCulling** | Skips rendering entities hidden behind walls |
| **MoreCulling** | Extends culling to leaves, crops, and more |
| **C2ME** | Parallel chunk generation and loading |
| **Voxy** | Level-of-detail rendering for distant terrain |
| **Dynamic FPS** | Limits FPS when the window is unfocused |
| **Debugify** | Fixes dozens of confirmed Minecraft bugs |
| **Supplemental Patches** | Additional targeted bug fixes |

## 🌅 Visuals

Three shader presets are included — pick the one that fits your hardware and mood.

| Shaderpack | Style |
|-----------|-------|
| **Complementary Reimagined r5.7.1** | Balanced, cinematic, feature-rich |
| **Solas Shader V3.4** | Warm and atmospheric |
| **I Like Vanilla v1.3.3b** | Subtle enhancement, minimal performance cost |

**EuphoriaPatcher** is included as a Complementary add-on for extended visual options.

Additional visual mods:
- **LambDynamicLights** - items and mobs emit real dynamic light
- **Continuity** - connected glass and textures
- **Entity Model Features + Entity Texture Features** - random/custom entity models and skins
- **Resource Packs Fusion** - allows multiple resource packs to merge assets seamlessly, preventing one pack from overriding another
- **Resource Packs Tweaks** - small enhancements to the resource pack system: better loading, custom settings per pack, and miscellaneous fixes
- **Particle Core** - particle rendering improvements
- **Sparkles** resource pack for enhanced particles

## 🧭 Quality of Life

Small improvements that add up to a noticeably better experience.

- **Jade** - informative tooltip when looking at any block or entity
- **AppleSkin** - hunger saturation and exhaustion HUD
- **Xaero's World Map** - full-world map accessible with `M`
- **Shulker Box Tooltip** - preview shulker contents on hover
- **Mouse Tweaks** - improved inventory drag behavior
- **WI Zoom** - smooth zoom with scroll wheel
- **Very Large Screenshots** - save screenshots at 4× or higher resolution
- **Armor Poser** - adjust armor stand poses in survival
- **Litematica + Schematic Preview** - build from schematics
- **Controlling** - search your keybind list
- **ModMenu** - in-game mod list and config access
- **No Resource Pack Warnings** - cleaner startup screen
- **Fix MC Stats** - corrects broken statistics tracking

## 🗺️ Included Datapacks

These datapacks are bundled and can be **enabled in your world**. They are entirely optional and work on any vanilla server that allows datapacks.

![The North Remembers](https://cdn.modrinth.com/data/NUznJSEG/images/b8c46614aac78d276fa82621bd4c2b93221b3526_350.webp)

### World Generation
| Datapack | Description |
|---------|-------------|
| **Tectonic** | Large-scale terrain overhaul — bigger mountains, deeper valleys |
| **CliffTree** | Adds trees growing on cliff faces for a wilder look |
| **CliffTree × Tectonic Compat** | Compatibility patch between the two above |
| **Nullscape** | Reworks the End dimension with new biomes and features |
| **Incendium** | Reworks the Nether with new biomes, structures, and mobs |
| **Dungeons and Taverns** | Hundreds of new structures across all three dimensions |

### Gameplay Tweaks
| Datapack | Description |
|---------|-------------|
| **Butterbee** | Butterflies - ambient passive mobs for lively environments |
| **No Enderman Grief** | Endermen no longer pick up and place blocks |
| **Players Drop Heads** | Players drop their head on death (great for SMP trophies) |
| **Invisible Item Frames** | Craft invisible item frames for clean wall displays |
| **Craft Light Blocks** | Adds crafting recipes for light blocks |
| **Stonecutting: Doors & Trapdoors** | Cut wood doors and trapdoors via stonecutter |
| **Stonecutting: Slabs & Stairs** | Cut wood slabs and stairs via stonecutter |
| **Too Expensive Extractor** | Work around the "Too Expensive" anvil limit |

## 📌 Notes

- The shaderpack you use is fully customizable — swap or disable them at any time in the video settings.
- Datapacks must be applied **before** or **at world creation** for world generation changes (Tectonic, CliffTree, etc.) to take full effect.
- This modpack does not modify any gameplay mechanics or add new items on its own. Everything stays vanilla-compatible.
